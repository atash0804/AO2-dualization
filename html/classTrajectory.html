◆ eliminate_incompatible()

uint64_t** Trajectory::eliminate_incompatible ( Coord  element)
inline

Create delta_ij set.

Includes elements of B incompatible with element. Suppose the element is B_ij. Incompatible elements are all B_kl such that either B_kj or B_il equals 1. To identify incompatible elements, we cycle throw rows k=1..n and look at B_kj. If it equals 1, all 1s of the row are incomatible. Otherwise, if B_kj = 0, the incomatible elements are all 1s from B_k & B_i (elementwise conjunction of two rows)

◆ find_neighbour()

bool Trajectory::find_neighbour ( )
inline

Find a neighbour for constructed trajectory.

Neighbour is the longest possible prefix of trajectory that can be used to construct a coverage. If the prefix ends with (B_t, Q_t), (B_t, Q_(t-1)) is appended to the neighbour.
